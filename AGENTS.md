# 教务材料生成系统 — 全域协作纪律

> 本文件定义跨 AI IDE 通用的绝对红线，所有 Agent 会话永久生效。

## 语言

- 所有指南、注释、计划、代码注释、输出说明使用**简体中文**。

## Agent 边界

- 每个 Agent 只修改自己工作区的文件。教务材料 Agent 只操作 `教务材料/`；课程工作区 Agent 只操作 `2025-2026-2 课程/`。
- 涉及对方工作区的操作须通过 `MSG_*.md` 消息委托，**不得跨区直接修改**。
- 跨 Agent 未达成共识的字段设计/架构变更，须通过 RFC 协商文档沟通。

## SSOT 原则

- **模板即 SSOT**：代码只做文本替换，不构造任何格式属性。修改格式 = 修改模板。
- **Schema-Spec-Data 三位一体同步**：修改任一层后须立即同步其余两层（`course_schema.py` / `Spec_*.md` / `Data_Dictionary.md`）。

## 文件管理

- **归档不删除**：过时文件移至 `_archived/` 或 `_deprecated/`，不直接删除。
- **目录干净**：活跃目录只保留必要文件。
- **输出隔离**：教案 → `教案/`，实验材料 → `实验材料/`，考核材料 → `考核材料/`。

## 文档规范

- 每个生成器一份 `Spec_*.md`，全局通用规则在 `Spec_Global.md`。
- 发现冗余文档时，将有效内容融入 Spec，原文件归档。
