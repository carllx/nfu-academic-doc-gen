## 1. 教学大纲撰写须知

### 1.1 保持材料统一
学业指南、教材、教学大纲、教案、课件必须严格保持统一（观测点对标25年普本人才培养方案）。
*   🔗 [参见学校要求](https://www.kdocs.cn/l/ctBI7KrDX6AI)

### 1.2 实行课程负责人制度
1.  **全权审核**：课程负责人全权审核及组织撰写以上材料，教学大纲上的任课老师签字仅由课程负责人签字。
2.  **集体授课**：需组建虚拟教研室（建议增加企业导师），[企业人员名单]需报备院教学指导委员会。
3.  **课程组会议**：
    *   **频次**：大纲、教案、AB卷、PPT制作等每个阶段都需要开会（至少两次线下）。
    *   **存档**：会议记录、签到等文件需存档并提交。
    *   🔗 [设计学院会议纪要模板](https://www.kdocs.cn/l/cog27TqtGsVy)
4.  **一门课一个大纲**：
    *   不同年级/层次（如普本、专升本）即使课程名称相同，也需根据人才培养方案使用**不同的教学大纲**。
    *   理论课、核心课作为基础课程，经审核后可保持一致，但实践课要完全不一致。
    *   **导师帮扶**：新入职老师在编写过程中实行导师帮扶制度。
    *   **作业差异**：同一门课程，因年级和培养对象不同，作业也应有所差异。

### 1.3 规范撰写
严格按照[学院提供的模板]进行撰写，注意字体、间距、内容规范。
*   🔗 [20260127（设计学院）附件1：课程教学大纲模板(1)](https://www.kdocs.cn/l/ce0O8O3hX1VA)

### 1.4 大纲内容注意（预评估专家意见）
*   **前沿融入**：融入人工智能、前沿技术，将OBE理念融入教学。
    *   🔗 [OBE理念参考](https://www.kdocs.cn/l/cm7Zs9jzyA9C)
*   **重点难点**：强化教学重点、难点与课程目标的对应关系。
*   **教材匹配**：坚持与选用教材深度匹配，教学内容体系与教材章节体系保持一致。一级、二级标题应与大纲对应，匹配度达到 **80%**。
*   **一致性**：教案与大纲中的教学方法、思政等必须一致（教案中的多次教学方法、思政等可以概述罗列于大纲中）。

### 1.5 理论与实践课时比例
*   **调整要求**：针对23级、24级人才培养方案理论及实践学时安排不合理（如0:40或0:60）的情况，要求参考25级普本人才培养方案或按实际授课比例填写（如20:20, 20:40, 30:30）。
*   **同步修改**：需同步修改实验指导书（实验学时总和 ≤ 实践学时，减去的学时在大纲中要有合理安排）。
*   **上限**：所有年级的理论课程实践学时不能超过18（如26:10）。

### 1.6 考核与成绩要求
*   成绩组成：**过程性考核** + **期末考核**。
*   🔗 [（发布版）251222设计学院课程考核成绩评定与分布规范](https://www.kdocs.cn/l/cvJ2mROhyq4P)

### 1.7 过程性考核（平时成绩）
1.  **形式**：分析/调研报告、实验、测试、演示PPT、论文、期中考核、大作业、课程作品、结果演示等。
2.  **要求**：考核累计 **不少于3种**，并明确每种分值占比。
3.  **实践**：必须包含对学生在实践教学学时中产出过程性成果作品的考核。

### 1.8 考试方式
1.  实践课外的必修课必须使用**考试**方式。
2.  选修课由任课教师决定采用考试或考查方式。

## 4. 实验材料

**需与教学大纲对应，进行实验材料撰写。**

### 📄 常用附件
*   🔗 [附件2：广州南方学院实验指导书（示例）](https://www.kdocs.cn/l/ckuz9yFto6MB)
*   🔗 [附件3：综合性、设计性实验项目认定表](https://www.kdocs.cn/l/cc2MGHGPOpLp)
*   🔗 [附件4：学生实验（实训、实践）报告（封面及内容模板）](https://www.kdocs.cn/l/cv4HmZw6JwGH)
*   🔗 [附件5：实验项目达成情况分析报告](https://www.kdocs.cn/l/ch0xMbr2AXEX)

### ⚠️ 注意事项
1.  **评分制**：实验报告成绩评分使用 **4级制**。
2.  **分数浮动**：平时作业与对应的实践报告分差、期末作业分差原则上 **不超过5分**（需按分工区和实际工作量给分）。
    *   *注：开学后第一周会有详细说明。*
3.  **类型要求**：实验报告类型 **不低于3种**（验证性、演示性、综合性、设计性）。
4.  **提交形式**：可以小组单位提交（1-5人/组）。
5.  **提交频次**：按阶段提交。
    *   例：1-18周课程，于第4、8、12、16周提交；1-9周课程，于第2、4、6周提交。
6.  **提交途径**：阿里云盘，并在共享表格中登记。

### 📌 实验类型说明
*   **演示性**：操作流程标准化，教师主导中心，全程主导操作与讲解。
*   **验证性**：学生为核心，自主操作，教师答疑纠错。针对具体步骤（如软件操作、临摹），需达到预设效果标准。
*   **综合性实验**：基于一定基础知识和技能，运用单门或多门课程知识进行综合训练。
*   **设计性实验**：学生自行设计实验方案、独立操作完成，并撰写综合分析报告。

## 8. 开学后执行要点

1.  **第一次上课**：**必须**将成绩构成、比例和评定依据详细告知学生。
2.  **随身携带**：授课时须携带教学大纲、教学进度表、教案、教材，以备检查。
3.  **每周必做**：
    *   学习通签到。
    *   缺勤/请假汇总（需安排班长/学委/课代表告知辅导员）。
    *   **及时登记**：出勤、过程性作业等情况。
